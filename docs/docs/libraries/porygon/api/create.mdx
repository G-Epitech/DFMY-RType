# `Create`

Create a new instance of `DependenciesHandler`.

```cpp
DependenciesHandler::Ptr Create();
```

## Parameters
- None

## Return value
- A pointer to the new instance of `DependenciesHandler`.

## Examples
```cpp
int main() {
  auto handler = DependenciesHandler::Create();

  // Use the handler

  return 0;
}
```
