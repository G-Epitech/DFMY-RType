# `GetShared`

Get a shared pointer on the current `DependenciesHandler` instance.

```cpp
DependenciesHandler::Ptr GetShared();
```

## Parameters
- None

## Return value
- A shared pointer to the current `DependenciesHandler` instance.

## Examples
```cpp
int main() {
  auto handler = DependenciesHandler::Create();
  auto ref = handler->GetShared();

  if (ref == handler) {
    std::cout << "The shared pointer is the same as the handler" << std::endl;
  } else {
    std::cout << "The shared pointer is not the same as the handler" << std::endl;
  }
  return 0;
}
```
